# Marfeel frontend test
A client side web application to show 3 charts with a fake server API. 
## Install
Steps to run the project

```
git clone https://github.com/irodriguez86/marfeel-test.git
cd marfeel-test
npm install
npm run dev
```

## Build for prod
```
npm run build
```
